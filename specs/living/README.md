# Living Specifications

本目錄描述系統「目前真實存在」的能力與約束，而不是本次準備修改的內容。

推薦分類：

- `architecture/`：系統邊界、元件、部署與信任邊界。
- `domains/`：各 Domain 的商業規則與狀態機。
- `contracts/`：API、Event、File、Data Schema 契約。
- `security/`：Authentication、Authorization、Audit、Encryption。
- `operations/`：SLO、監控、備份、DR、Runbook。

每次 Change Closure 前，必須將核准變更同步至 Living Specs。
