## 工具命令

- Spectra：discuss／propose／apply／ingest／archive
- Alfredo：auto／specify／plan／tasks／implement／resume／drift／adr
- Mifkata：spec-create／compact／apply／verify／refactor／test-create

## 公司原則

- AI 不核准商業、架構、安全與 Production 風險
- Code → Spec 必須有決策證據
- Archive 前必須 Test／Drift／Living Sync
